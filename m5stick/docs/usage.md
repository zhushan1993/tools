# 使用说明

## 快速开始

### 1. 启动 Bridge

```bash
cd m5stick
./setup.sh                          # 首次：搭建环境
bridge/venv/bin/python -m bridge.server  # 启动服务
```

### 2. 授权确认流程

当 Claude Code 需要执行操作（如 Bash 命令、写文件）时：

1. M5STICK 屏幕会显示命令详情和工具名称
2. 屏幕底部显示 **确认(A)** / **拒绝(B)**
3. 按 **A 键** → 允许执行
4. 按 **B 键** → 拒绝执行
5. 30 秒无操作 → 自动拒绝

### 3. 语音输入

说话前确保 M5STICK 处于待机界面（显示 "Claude Code" 和 "长按 B 键说话"）：

1. **长按 B 键** (>1s) → 屏幕切换为录音界面
2. 对着 M5STICK 顶部麦克风说话
3. 说完 **按 B 键停止**，或 15 秒自动停止
4. M5STICK 显示 "识别中..."
5. Bridge 将识别出的文字注入 Claude Code

### 语音提示：

- **简洁** — 一次说出完整的需求，如"列出当前目录"、"创建一个 Python 文件"
- **清晰** — 正常语速，避免含糊
- **环境** — 安静环境识别率更高（PDM 麦克风拾音范围约 30cm）

## 注意事项

- **电池续航**：满电约 4 小时待机/1 小时连续使用，Type-C 充电
- **蓝牙距离**：最佳 5 米内，穿墙可能不稳定
- **首次配对**：需在电脑蓝牙设置中手动配对一次，之后自动连接
- **断开重连**：M5STICK 意外断开后会自动重连
